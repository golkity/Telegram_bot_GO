## Структура проекта
```bash
project/
├── main.go                # Точка входа приложения
├── config/
│   ├── config.go          # Конфигурация приложения (чтение токена, базы данных и других настроек)
│   └── config.yaml        # Файл настроек (токен бота, настройки API, пути и пр.)
├── handlers/
│   ├── messages.go        # Обработчики текстовых сообщений
│   ├── commands.go        # Обработчики команд (/start, /help)
│   └── callbacks.go       # Обработчики callback-запросов от inline-кнопок            
├── keyboards/
│   ├── inline.go          # Inline-клавиатуры (динамические кнопки)
│   ├── reply.go           # Reply-клавиатуры (кнопки быстрого ответа)
├── models/
│   ├── user.go            # Модели пользователей и их данных
│   └── session.go         # Модели сессий или временных данных
├── storage/
│   ├── db.go              # Логика подключения и работы с базой данных
│   └── migrations.sql     # SQL-скрипты для инициализации базы данных
└── go.mod                 # Зависимости 
```
